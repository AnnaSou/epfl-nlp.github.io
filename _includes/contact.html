    Application

    Interested in doing a project with the NLP lab? First, select one or more projects you're interested in from the list below (don't forget to check their full description on the linked document, where you'll also find the supervisor's email). Then send an e-mail to the project(s) supervisor(s) as well as to:
    nlp-projects-apply@groupes.epfl.ch

    Please attach your CV and transcript and include [Masters Project] or [Masters Thesis] in your subject heading. Looking forward to collaborating with you!

    Open projects

    Title Supervisor Short Description
    Semantic pointers in transformers Gail Weiss Take a trained LLM and see if semantic pointers information can be found in the individual embeddings of the tokens of a given sequence, and moreover at which layers. Can seek with both rich probe (eg 2 layer MLP) to see if information is present at all, and simple one (linear transformation) to see how ‘simply’ it is represented.
    Knowledge-aware change captioning in remote sensing images Syrielle Montariol Build a multimodal dataset describing events with an impact visible from the sky, using news articles and satellite images. Then, use large language models to generate description of the geographical changes occurring in the images, using knowledge from the news articles.
    De-embedding as Analysis Gail Weiss Compare ‘informativeness’ of embedding space projection proposed by paper to other potential projections - potentially validating it as a good projection to consider or alternately showing alternative interpretation methods.
    Bridging the Gap between natural and remote sensing images in large Vision-Language models Syrielle Montariol Large Vision-Language models (VLMs) are trained on large amounts of interleaved text-images data. However, they see very few satellite images and are often unable to perform tasks related to them. The goal is to equip generic VLM with remote sensing understanding ability and cross-modality (RS images and natural images) understanding ability, using existing aligned datasets between remote sensing and natural images.
    Query and Key Clustering Analysis Gail Weiss See if we can get a symbolic representation of the queries and keys used by a specific attention head. Then, see if we can interpret this presentation, giving it meaning and managing to predict what it will be for the different positions at new inputs. See if we can build a method - probably with human analysis as part of it - to do this for multiple attention heads, one at a time.
    Skill extraction and matching from heterogeneous data sources Syrielle Montariol Existing models aiming at recognizing education-related entities (skills and competencies) are often limited to one kind of data source (resumes, job postings or course description), one domain (technology, management...), and one language. The goal is to transfer these models and methods across languages, domains and sources. This project aims at exploring the usage of large language models to perform this transfer, through methods such as in-context learning or using external knowledge bases.
    NMT - Translated Token Preparation by Encoders Gail Weiss Test various hypthoses on whether/how well NMT encoders predict/prepare the output tokens that the decoder will use in its translated output, and if so, how the decoder interacts with these predictions.
